# 一个四川省中小学智慧教育AI培训自动学习浏览器插件

# 使用说明

- 打包下载源码
- 在Chrome和Edge浏览器插件扩展页面开启开发者模式，然后加载该插件
- ![alt text](image-2.png)
- ![alt text](image-3.png)
- 进入某个课程详情页，点击右上角【一键学习】耐心等待即可，学习完成后会自动刷新页面，只需要点击一次，无需重复点击。
